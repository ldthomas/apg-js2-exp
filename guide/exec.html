ApgExp.exec(input)

The exec() method executes a search for a match in a specified string. Returns a result object, or null. This string may be specified as a JavaScript string or an array of integer character codes.

Syntax

var exp = new ApgExp(pattern, flags);
var result = exp.exec(str);

Parameters

  • str
    • string - The string to search for a pattern match in.
    • array - An array of the integer character codes of the string to search for a pattern match in.

Return

If the match fails, the return is null. Otherwise, the return is a result object and properties of the ApgExp object are updated. The result object contains the phrases matched to all of the rules named in the pattern syntax.

Examples

var exp = new ApgExp(pattern);
var result = exp.exec(string);

With the g or y flag set, a successful match will increment lastIndex enabling multiple calls on the same string. 

var pattern = 'AsAndBs = "a" *"b"\n';
var str = "abbcccabxx";
var exp = new ApgExp(pattern, "g");
while(true){
var result = exp.exec(str);
  if(result){
    console.log("found: " + result[0] + " :at: " + result.index);
  }else{
    break;
  }
}
/* returns */
found: abb :at: 0
found: ab :at: 6

Note that patterns, in general, can match empty strings (as, in fact, can regular expressions). To be safe, a loop as above should be used instead of the more compact, but possibly infinite: 

var exp, result;
exp = new ApgExp(pattern, "g");
while(result = exp.exec(str)){
    console.log("found: " + result[0] + " :at: " + result.index);
}
